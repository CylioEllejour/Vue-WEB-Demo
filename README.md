## 环境需求 Requirements

使用```npm```命令安装一下库

```npm
npm install express
npm install cors
npm install mysql
npm install axios
```

## 安装 Installation

- Clone the project

> git clone

- Run server
  
> cd server
> node app.js

- Run WEB

> npm run serve

- Start

复制终端的web网站到浏览器中，enjoy项目吧！:)
